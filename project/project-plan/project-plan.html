Site Name

I picked Weather Teller for my website because it's a weather site and it will tell the user what the forecast will be like as well as the current weather for any specific area.

Purpose

The purpose for this site will be to help users know about the weather forecast and the current weather, so they can make plans accordingly. They can also see what the weather is like for pretty much anywhere in the world if they're curious.

Scenario Questions

  • How would I be able to pick an area to view the weather forecast on?
  • Where can I find the history of my past searches?
